List接口

  • ArrayList和Vector使用该数组实现
  • LinkedList使用了双向循环链表的数据结构，随机存取性能较差，每次获取都需要遍历整个链表
  • ArrayList的任意位置插入和删除是消耗性能的，会导致底层数组的重组。
  • 默认情况下，ArrayList的数组大小的初始值时候10，每次扩容将新的数组大小设置为原来大小的1.5倍。
