Java内置工具

JPS(Java Virtual Machine Process Status Tool)

显示当前所有java进程的pid的命令，JPS工具是在JAVA_HOME/bin目录下，比较常用的参数如下：

  • jps -v 输出传递给JVM的参数，一般较为齐全
  • jps -l 输出完整的包名和完整的jar

备注：只能显示当前用户的所有java进程

JSTACK(Java Stack Treace)

显示给定的Java进程id的或java远程调用服务的java堆栈信息

  • jstack -l pid

JSTAT(Java Virtual Machine Statistics Monitoring Tool)

  • 类的加载和卸载情况
  • 查看新生代、老年代和持久代的容量以及使用情况
  • 查看新生代、老生代和持久代的垃圾回收情况
  • 查看新生代中Eden区及Survior区中容量及分配情况等

常用命令：

  • jstat -class pid:查看当前类加载和卸载情况
  • jstat -gc pid:查看当前内存垃圾回收情况

详细使用文档http://docs.oracle.com/javase/1.5.0/docs/tooldocs/share/jstat.html

JMAP(Java Memory Map)

打印java进程的内存的，所有对象的情况。

  • jmap -heap pid 查看Heap配置信息以及Heap使用情况
  • 查看当前类-histo[:live] 打印每个class的实例数目,内存占用,类全名信息. VM的内部类名字开头会加上前缀”*”. 如果live子参数加上后,只统计活的对象数量.
  • 生成dump文件,供MemoryAnalysisTool分析使用

JINFO(Java Configuration Infomation)

输出java运行参数以及命令行参数

JHAT(Java Head Analyse Tool)

用来分析java堆的工具，可以将堆的中对象以HTML形式展现出来，包括数量，大小并支持对象查询语言。

  • 导出堆
  • 分析堆

  • 查看HTML

  • jhat中OQL语句
