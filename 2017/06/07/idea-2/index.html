常用的代码提示

  • ctrl+j可以查看所有的快捷提示代码

Java

快捷提示 作用
psvm 生成Java类的Main方法
serr System.err.print()
sout System.out.print()
soutm System.out.println(“Test.main”);打印类名和方法名称
soutp System.out.println(“args = [“ + args + “]”);打印方法参数
soutv System.out.println(“args = “ + args);打印变量
psfs public static final String;公共静态字符串常量
psfi public static final int;公共静态整数常量

markdown

快捷提示 作用
.image 图片链接
.table 表格
.link url链接
.codefence 代码

添加自定义的代码提示

  • Settings->Editor->Live Template

  • 先添加Template Group,再添加Live Template

  • 添加PI(私有整数了类型成员变量)

  • 注意红色框表示的是代码提示用于的范围，比如pi只用于java范围
