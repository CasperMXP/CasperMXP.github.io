如何关闭issue

上面新建后issue的状态一直都是open,也就意思是bug或功能一直没有修改完。当你修改完了，测试通过后，你就应该关闭改issue了。
这个只需要在git commit -m 后加上一句close issue编号。push之后，你就会发现此时issue就已经是close状态了。
